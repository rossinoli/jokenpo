# Jokenpô Android ✊✋✌️

Jogo de Jokenpô (Pedra, Papel ou Tesoura) desenvolvido em Java para Android.

---

## 🚀 Funcionalidades
* Escolha entre **Pedra**, **Papel** ou **Tesoura**.
* Oponente (Android) com escolha aleatória.
* Verificação automática de **Vencedor**, **Perdedor** ou **Empate**.
* Interface simples e intuitiva.

---

## 🛠️ Tecnologias Utilizadas
* **Linguagem:** Java
* **Interface:** XML (ConstraintLayout)
* **Ferramenta:** Android Studio
